**Values:**

* **DISABLED** -Disabled called successfully on reader

* **ENABLED** -Enabled reader successfully

* **SCANNING** -Successfully started scanning

* **IDLE** -Finished scanning

* **ERROR** -Error occurred

* **UNKNOWN** -Unknown status
