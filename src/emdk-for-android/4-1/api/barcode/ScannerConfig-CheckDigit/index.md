The number of check digits to be verified. With MSI symbols, one check
 digit is mandatory and always verified by the reader. The second check
 digit is optional.
 
 

**Example Usage:**
	
	:::java
	
	CheckDigit.ONE;
	


**Values:**

* **ONE** -Verify one check digit.

* **TWO** -Verify two check digits.
