All the version keys for  VersionManager#getVersion(VERSION_TYPE) are enumerated here.
 
 

**Example Usage:**
	
	:::java	
	VERSION_TYPE.EMDK


**Values:**

* **EMDK** -EMDK version

* **MX** -MX framework version

* **BARCODE** -Barcode Scanner Framework version.
