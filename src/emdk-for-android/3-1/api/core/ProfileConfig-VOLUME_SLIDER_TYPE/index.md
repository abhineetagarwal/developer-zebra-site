Defines the notification type for a good decode.

 

**Example Usage:**
	
	:::java	
	VOLUME_SLIDER_TYPE.DEFAULT;


**Values:**

* **DEFAULT** -Default prefix configured on device

* **RINGER_AND_NOTIFICATIONS** -Ringer and Notifications.

* **MUSIC_AND_MEDIA** -Music and Media.

* **ALARM** -Alarms.
