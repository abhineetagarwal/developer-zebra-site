The data type contained within the region

**Values:**

* **UNKNOWN** -Data type is Unknown.

* **OCR** -Data type is OCR.

* **OMR** -Data type is OCR.

* **BARCODE** -Data type is Barcode.

* **PICTURE** -Data type is picture.
