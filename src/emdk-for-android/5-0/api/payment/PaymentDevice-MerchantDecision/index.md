The merchant decision notes additional handling for the EMV request based
 on required processor handling.

**Values:**

* **REQUEST_TC**

* **FORCE_ONLINE** -This merchant decision insists the transaction to be online with the usage of PIN.

* **FORCE_DECLINE**
