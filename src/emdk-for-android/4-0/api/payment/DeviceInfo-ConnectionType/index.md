List of supported payment connection types.
 
 

**Example Usage:**
	
	:::java
	
	ConnectionType.BLUETOOTH;
	


**Values:**

* **BLUETOOTH** -The payment device is connected via Bluetooth to the mobile device.

* **UNDEFINED** -Connection type is not defined.
