Defines all the error codes the diagnostic API returns.
 <br><br>

**Values:**

* **SUCCESS** -Success.

* **FAILURE** -Failure

* **FEATURE_NOT_SUPPORTED** -Feature not supported.

* **INVALID_VALUE** -Invalid value

* **INVALID_OBJECT** -Invalid Object

* **UNDEFINED** -Error code not defined.
