List of supported scanner device type.
 
 

**Example Usage:**
	
	:::java	
	SimulScanDeviceType.CAMERA;


**Values:**

* **CAMERA** -The Scanner device class is camera.

* **IMAGER** -The scanner device class is imager.

* **UNDEFINED** -The scanner device class is not defined.
